# Instaskip

Transforms from eskip to json to eskip.

## Installation

Download from https://clojars.org/org.clojars.danpersa/instaskip.

For maven users

    <dependency>
      <groupId>org.clojars.danpersa</groupId>
      <artifactId>instaskip</artifactId>
      <version>0.1.0-SNAPSHOT</version>
    </dependency>

## Publishing
